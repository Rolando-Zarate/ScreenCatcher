ScreenCatcher

A application by Rolando Zarate

What is ScreenCatcher?

ScreenCatcher is a desktop app for
take screenshots with 3 formats (png, jpg, bmp).
ScreenCatcher is open source, using Apache License v2.
Available for Microsoft Windows and Linux.

Requirements:

For running the pure python version in Linux/Microsoft Windows you need Python 3.8+ with these modules: pyautogui, tkinter, os, time.
In Microsoft Windows you can download the EXE below.
In Linux you need to use Python interpreter 3.8+ with the source code or bytecode.
